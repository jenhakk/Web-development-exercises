Differences between HTML and HTML5

The main differences between HTML and HTML5 are that with the HTML5 there is no need for plugins
when adding video and audio to pages, data can be stored locally which enables apps to continue working offline
and overall it simplifies creating web applications and ensures smooth and consistent experience for
web users and developers as well. Also, the HTML5 has improved error handling, many new elements (like section,
header and article) and it supports custom data attributes.
